# AI Usage

ChatGPT was used to help plan, generate, and debug this project.

AI helped with:
- Planning the Housing Roommate Finder idea
- Designing the PostgreSQL schema
- Creating backend routes with Express
- Adding login/logout
- Hashing passwords with bcrypt
- Adding JWT authentication
- Adding employee, HR, and admin roles
- Creating a React frontend
- Organizing the project into client and server folders

Security best practices:
- Passwords are hashed
- JWT tokens are used for login
- Protected API routes require authentication
- Role-based access control protects HR/admin pages
- Database credentials are stored in .env
- SQL uses parameterized queries
